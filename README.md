# Learning-Resource
团队学习资源整理
## 前言
>对于马上退队的大三狗,没有太多可以留下的,只能够将自己所学和经验进行分享.也希望创数据团队会越来越好!
### 项目简介
该项目主要用于对团队学习的资源进行整理和分享,提高团队的学习效率,提供更清晰的学习路线.
希望新老队员都能积极参与,将自己的学习资源和经验进行分享.
### 学习经验
对于每一个团队的新成员,我希望不要盲目的开始学习.搞清楚以下三点:
* [目前什么技术栈最适合自己学习?](学习经验-Q1.md)
* [怎么学习最快?](学习经验-Q2.md)
* [怎么总结和复习?](学习经验-Q3.md)
